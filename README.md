# Refolio

## すぐ試す方法

1. `start-purrboard.bat` をダブルクリック
2. 依存関係が未インストールなら自動で `npm install`
3. 開発サーバーが起動し、ブラウザで `http://127.0.0.1:4173` が開きます

## ターミナルから起動する場合

```powershell
cd "C:\Users\araya\Desktop\作業フォルダ\Codex\purrboard-mvp"
npm run dev
```

## うまくいかないとき

- Node.js が入っているか確認: `node -v`
- npm が入っているか確認: `npm -v`
- ポート 4173 が使われている場合は、起動中のサーバーを止めてから再実行
